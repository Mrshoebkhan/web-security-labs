# Web Security Labs

This repository contains hands-on web application security lab work
performed in authorized practice environments.

## Labs Covered
- OWASP Top 10 vulnerabilities
- DVWA and OWASP Juice Shop testing
- Manual testing using Burp Suite

## Content Format
- Markdown write-ups
- Screenshots (sanitized)
- Step-by-step testing approach

> ⚠️ All activities are performed on legally authorized lab environments only.
